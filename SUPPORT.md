# Support

Agent Permit Office is currently an open-source MVP project.

## Community Support

For public project usage after launch:

- use GitHub issues for bugs, false positives, rule requests, and docs gaps
- include command output, artifact paths, and relevant rule IDs
- remove secrets, customer names, private repository data, and raw traces before posting

## Commercial Support

The planned open-core business keeps managed workflow and enterprise support outside the free core.

Potential paid support areas:

- private repo onboarding
- custom rule packs
- policy-pack design
- CI rollout help
- hosted control plane
- SSO/RBAC and audit evidence retention
- enterprise response SLA

No commercial support channel is active yet.

## Not Supported Yet

Not supported in the current MVP:

- hosted dashboard
- production SLA
- self-hosted enterprise package
- security incident response retainer
- private vulnerability bounty
- guaranteed model-provider uptime
