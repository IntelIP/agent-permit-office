# Roadmap

Date: 2026-06-07

PermitGraph is moving from local MVP proof to public open-core launch.

## Now: Public Core Hardening

- publish-ready governance files
- normal CI for tests and self-scan
- sanitized demo artifact strategy
- release checklist
- first tagged release plan

## Next: Community Proof

- public repo launch
- sanitized demo report
- rule contribution guide
- false-positive issue templates
- curated public repo validation set
- lightweight release notes

## Later: Team Workflow

- hosted dashboard prototype
- GitHub app integration
- scheduled scans
- shared policy profiles
- team approval queue
- report history
- managed model gateway option

## Enterprise Direction

- SSO/RBAC
- audit evidence retention
- custom rule packs
- self-hosted or VPC deployment option
- procurement/security review packet
- support SLA

## Explicit Non-Goals For MVP

- no dynamic execution sandbox
- no automatic MCP execution
- no hosted customer data storage
- no unbounded Deep Agent repo browsing
- no AI-only permit decisions
